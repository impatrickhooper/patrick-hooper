WGI Informer

{ OVERVIEW }

Wantman Group, Inc. (WGI) had a dated intranet running on a difficult-to-use Drupal setup with a nightmare for a codebase. They gave me the opportunity to build them a new one, so I gave them a clean, user-friendly site with more bells and whistles than they could have imagined!

Roles:
Logo, UI, UX, Full-Stack

Technologies:
WordPress + Underscores, Materialize, Font Awesome, Masonry, Mapbox, Typeform, Sass, Breakpoint

Platforms:

GitHub

{ HOME }

The home page provides highlights of the most important information from around the intranet and within the company. This means a hero slider for big announcements, event cards (when there are some upcoming), introductions to new employees, employee birthdays and work anniversaries, news cards, and some photos albums from recent events.

{ NAVIGATION }

There are a few flavors of navigation that can be found on the intranet. The first is the left side slide-in navigation accessed via a hamburger/menu icon. Next are the Favorites, accessed via a star icon that opens a dropdown drawer. The Spotlight section contains some fun, random snippets of info and is opened as a slide-in bottom modal using a lightbulb icon. Finally, there are floating action buttons fixed in the bottom right corner that open various Typeforms.

{ EVENTS }

Past, present, and future events are displayed in a 2 column card form and grouped by the year in which they took place. Enough details are given to wet your whistle (and an event teaser photo), at which point you can click through and see the full event page if you'd like to know more.

{ NEWS }

Pieces of company news are displayed all fancy using Masonry and grouped by the year in which they were posted. If there's a teaser, you can click on the card photo to reveal it, or you can click the card link to go to the full new article. From there, you can cycle through all News posts using the post navigation.

{ OFFICES }

Mapbox was used to create an awesome, interactive map for all of the WGI offices. Naturally, you can click a marker to get more information about each office. You can also use the filter to see what services are offered at each location! Locations too near to each other are clustered to avoid frustration that stems from markers overlapping; simply click a cluster to zoom in a little more and make your life easier.

{ PEOPLE }

The company directory is a big, colorful experience that allows you to creep on your peers at your own pace. Interested in an extremely immersive experience? Great, go page-by-page and soak it all in. Really just here to look for a specific person? We've got you covered - use the search to find what you need quickly and expand a person's card or click through to their profile for more info.

{ PHOTOS }

Photos from company events are stored in Google Photos and linked to from this page. Each album gets a thumbnail photo and name that links out to the respective Google Photos album. These are presented to users in a simple, clean thumbnail grid. For an added bonus, the active/hovered over album appears brighter and with a slight zoom!

{ PROFILES }

Every user has a unique profile and account that they can customize as they see fit. It's accessed using the link next to their photo in the main side navigation, and presented in a clean, appealing format.
