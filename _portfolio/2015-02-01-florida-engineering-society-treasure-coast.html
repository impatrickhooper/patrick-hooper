{ HOME }

The home page was used to showcase the most important information from TCFES: upcoming events, news, and chapter photos. Chapter photos fade in as they come into focus for a little something extra.

{% include browser.html url="tcfes.org" image="portfolio_florida-engineering-society-treasure-coast" extension="jpg" alt="| Home" %}

{ NEWSLETTER }

Since 2002, TCFES has regularly created and sent out a newsletter. Previously they were posting these as lists of links with super old files zipped up into one archive. The organization and presentation of these links was reformatted into vertical tabs for each year, making all those newsletters accessible in an easy-to-use, digestable format.

{% include browser.html url="tcfes.org/newsletter" image="portfolio_florida-engineering-society-treasure-coast_newsletter" extension="jpg" alt="| Newsletter" %}

{ BOARD MEMBERS }

TCFES board members and their information were previously displayed in a straight HTML table. Obviously, that changed. Members with available pictures got a small logo overlay representing their college, and those without just got a big logo. Information was presented in a clean, simple grid.

{% include browser.html url="tcfes.org/board-members" image="portfolio_florida-engineering-society-treasure-coast_board-members" extension="jpg" alt="| Board Members" %}

{ EVENTS }

TCFES wanted a better event tracking system. The solution gave them that, allowing them to easily input event details, save locations, and search based on various fields/location.
