{ HOME }

wgi.buzz needed a way for users to easily access the various tools available to them. The solution was to organize those tools by category on a splash page. For a little extra flair, each category was given a photo. When that category is clicked, the photo slides away to reveal links to the resources.

wgi.buzz

{ E-FORMS }

The original reason wgi.buzz was started was to reinvent a paper form as an online form fit for the 21st century. The specific form was called the Project Initiation Form and manifested itself as a Google Apps Script form with conditional fields, dynamically created fields, and file uploads. Upon submission, it writes to a Google Sheet, is electronically transmitted to the approver, and then forward to the Accounting division.

wgi.buzz/e-forms

{ OPPORTUNITIES }

WGI needed a method to track its efforts trying to bring in new work. Thus, the Opportunities section of wgi.buzz was born. It places each project and its 3 phases on a timeline. Users can hover over each phase to see more information. It also has a filter so project mangers can zero in on their division. The data is housed in a collaborative Google Sheet accessed by the Google Drive API.

wgi.buzz/opportunities

{ SALES GOALS }

WGI sets annual sales goals for its project managers, but until now had no reliable, centralized system for tracking performance against the goals. Using an easy-to-update Google Sheet and the Google Drive API to feed a chart that would (hopefully) make United Way proud, users can now view goals by location and interact with the chart to see the current sales and goal.

wgi.buzz/sales/goals
