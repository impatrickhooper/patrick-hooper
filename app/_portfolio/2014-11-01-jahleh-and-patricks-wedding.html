{ OUR STORY }

We really wanted to emphasize how we got here and how much fun we've had along the way in a creative, visual way. So we pulled together a bunch of our favorite pictures to help us out, wrote some (clever?) taglines, and played connect the dots.

{% include browser.html url="wedding.hoopr.io" image="portfolio_jahleh-and-patricks-wedding" extension="jpg" alt="| Our Story" scroll="scroll" %}

{ WEDDING }

We needed a spot to provide actual details about the wedding to guests (kind of important, I guess). So we put that all here, including when, where, some pretty photos, and interesting travel information about the location.

{% include browser.html url="wedding.hoopr.io/details" image="portfolio_jahleh-and-patricks-wedding_wedding" extension="jpg" alt="| Details" scroll="scroll" %}

{ RSVP }

Each invitation sent out had a personalized code attached. The invitee could then enter this code in the RSVP form to bring up their name and the name of their +1, where applicable. To finalize their RSVP, it was as simple as checking the box if they were attending and then clicking submit. Easy breezy.

{% include browser.html url="wedding.hoopr.io/rsvp" image="portfolio_jahleh-and-patricks-wedding_rsvp" extension="jpg" alt="| RSVP" scroll="scroll" %}

{ GUESTBOOK }

For a little extra fun, we added an online guestbook to our site. Just fill out the form, click sign, and your name and message are forever embodied in our digital wedding archive. Sweet deal.
