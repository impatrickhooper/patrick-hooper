{ HOME }

The home page lists each division of WGI and its associated color in a stacked card-style format. Clicking a card will take you to the page for that division. At the bottom of the home page (and every page) is a footer containg the names of the content managers for each division.


{ NAVIGATION }

The WGIki has a few navigational components. A menu icon opens the left side slide-in nav with some user info as well as division links. Categories for a division appear if you're under that division's umbrella of pages. Clicking the star icon opens the Favorites drawer with quick links. Finally, fixed in the bottom right corner are floating action buttons that open for feedback or help forms.

{ DIVISIONS }

At the top of the resource hierachy is the division. Each division within WGI has its own categories and resources that it maintains. On the main page for each division, links to all its categories are listed with a brief description. Oh, and just for a little added flare, each division also has a unique color that is used thematically throughout its pages (even the nav bar).


{ CATEGORIES }

Next up in the hierachy are categories. Individual resources are assigned categories, and the main page for each category lists all the resources that make it up. A short description is provided to help give users an idea of what they'll find here. Resources are presented in a tab-like list with icons and other useful information to let users know what they're in for. Hovering over a resource makes it scale up (slightly).

{ RESOURCES }

Finally, at the bottom of the hierachy are individual resources. These can come in a variety of formats (single files, multiple files, external links, self-composed articles, etc.), and an edit button is available to content managers for quick access to the backend of a page.


{ SEARCH }

The search functionality of the WGIki scours all resources and returns a list of the most relevant ones (using Relevanssi). The results are listed just like resources on a category page, with a little additional information about where to find them specifically.
